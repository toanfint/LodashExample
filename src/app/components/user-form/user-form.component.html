Form dirty? {{ userForm.dirty }}

Form value thay đổi (so với dữ liệu gốc)? {{ isChanged }}

userForm.dirty chỉ cần input thay đổi value một lần là true, kể cả khi bạn gõ rồi sửa lại về giá trị ban đầu

_.isEqual(value, originalData) giúp so sánh nội dung thực sự với dữ liệu gốc.

Nhờ đó, nếu người dùng nhập lại đúng dữ liệu ban đầu → isChanged sẽ là false.
